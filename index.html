PRIMEROS PASOS EN HTML - CSS

HTML

HTML (Lenguaje de marcado de hipertexto o Hyper Text Markup Language por sus siglas en ingles) es un lenguaje descriptivo que especifica la estructura de las páginas WEB.

CSS

Hojas de estilo en cascada (del inglés Cascading Style Sheets) o CSS es el lenguaje de estlos utilizado para describir la presentación de documentos HTML o XML.
